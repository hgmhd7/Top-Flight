A global study of correlations in airport ratings...

Correlations Between Airport Ratings and Various Factors

​

This D3 application is a study of over 2,000 airports around the world. We take a look at different factors that could possibly influence and/ or corrlate with the ratings of airports around the globe. Please explore the data by clicking the different factors on the plot to see if you can spot any possible outliers, correlations, and anomalies. Which factors have the highest correlations? What comparrisons produce unexpected results? Which factos confim a pre existing cinclustion you already had? You can examine these and many more questions with this interactive plot.

The data for this app was obtained from the OpenFilghts.org data base. Visit the Open Filghts site here. Supporting data was obtained by utilizing Google Places API calls to get information such as airport ratings, airport landing page (pun intended...), and number of ratings per data point. You can learn more about Google's API services here. Also, if you wish to take a look at the code for this project, visit my GitHub page that contains all of the original code for the project.
